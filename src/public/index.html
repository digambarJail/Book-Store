API Routes

Input: Get all Users

Route: https://book-store-84io.onrender.com/api/getUsers

Example: https://book-store-84io.onrender.com/api/getUsers

Input: Get all Books

Route: https://book-store-84io.onrender.com/api/getBooks

Example: https://book-store-84io.onrender.com/api/getBooks

Input: BookName or term in bookname

Route: https://book-store-84io.onrender.com/api/getBooks?search=(bookname)

Example: https://book-store-84io.onrender.com/api/getBooks?search=gar

Input: Rent Price range

Route: https://book-store-84io.onrender.com/api/getBooks?range=(from-to)

Example: https://book-store-84io.onrender.com/api/getBooks?range=5-10

Input: Category + name/term + rent range

Route: https://book-store-84io.onrender.com/api/getBooks?category=(category)&search=(search)&range=(range)

Example: https://book-store-84io.onrender.com/api/getBooks?category=fiction&search=the&range=2-4

Input: BookName + User Id + IssueDate to issue a book

Route: https://book-store-84io.onrender.com/api/issueBook?doi=(yyyy-mm-dd)&uid=(1,2,3,4,5)&bookName=(bookName)

Example: https://book-store-84io.onrender.com/api/issueBook?doi=2024-09-10&uid=3&bookName=Investing%20101

Input: BookName + User Id + ReturnDate to return a book

Route: https://book-store-84io.onrender.com/api/returnBook?dor=(yyyy-mm-dd)&uid=(1,2,3,4,5)&bookName=(bookName)

Example: https://book-store-84io.onrender.com/api/returnBook?dor=2024-09-11&uid=3&bookName=Investing%20101

Input: BookName to return past and current issuers + total rent by that book

Route: https://book-store-84io.onrender.com/api/getIssuers?bookName=(bookName)

Example: https://book-store-84io.onrender.com/api/getIssuers?bookName=Investing%20101

Input: User id to get all books issued to that person

Route: https://book-store-84io.onrender.com/api/getBooksByUser?uid=(1,2,3,4,5)

Example: https://book-store-84io.onrender.com/api/getBooksByUser?uid=3

Input: Date range

Route: https://book-store-84io.onrender.com/api/getBooksByDate?from=(yyyy-mm-dd)&to=(yyyy-mm-dd)

Example: https://book-store-84io.onrender.com/api/getBooksByDate?from=2024-09-01&to=2024-09-10
